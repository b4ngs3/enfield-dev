Web Development Courses

Whether you are just starting out or looking to level up your skills, these resources will help you on your journey to becoming a web developer.

Free Learning Resources

These are completely free and a great place to start if you are new to web development.

W3Schools

HTML & CSS Tutorial

The most popular reference site for web developers. Covers HTML, CSS, JavaScript, PHP and much more with interactive examples and exercises.

Visit W3Schools →
Codecademy

Learn HTML & CSS

Codecademy offers structured, interactive courses on HTML and CSS. Great for beginners who want hands-on practice with immediate feedback.

Start Learning →
YouTube – freeCodeCamp

HTML & CSS Full Course (6hrs)

A comprehensive 6-hour video tutorial covering everything you need to know about HTML and CSS from absolute beginner to confident developer.

Watch on YouTube →
freeCodeCamp

Responsive Web Design Certification

A free, self-paced certification covering HTML, CSS and responsive design. Earn a recognised certificate upon completion of all projects.

Get Certified Free →
The Odin Project

Full Stack Web Development

A free, open-source curriculum for learning full-stack web development. Covers HTML, CSS, JavaScript, Git, and more through real projects.

Start the Curriculum →
MDN Web Docs

Web Development Guides

Mozilla's official documentation and learning guides for web technologies. The go-to reference for HTML, CSS and JavaScript for developers worldwide.

Read the Docs →
